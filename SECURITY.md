# Seguridad — RouteFleet

- **JWT** para autenticación de repartidores y oficinas
- **Firmas digitales (POD):** capturadas con Canvas nativo, embebidas en PNG, sin almacenarse en servicios externos
- **OCR:** Tesseract.js (local en el navegador) — ningún dato de albarán sale del dispositivo
- **API:** CORS configurado, HTTPS forzado (Render + GitHub Pages)
- **Sin secrets** en el repositorio
